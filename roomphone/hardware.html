Supported Hardware

We're always looking for the next device to support, but for the moment, Room Phone supports a limited number of devices from ClearOne and Biamp. Please read below for specific device support and what information is required to connect to the device with Room Phone.

ClearOne

Currently, we support ClearOne CONVERGE Pro devices that include a telephone interface.

In order for Room Phone to connect to our telephony-equipped ClearOne device, you will need the following information:

  • IP Address
  • Username
  • Password

In larger systems the following information may also be required:

  • Device Type
  • Device ID

Biamp

Currently our support for Biamp devices is limited to devices with the VoIP-2 card.

  • IP Address
  • VoIP Block ID
  • Transmit Block ID
  • Receive Block ID
  • Device ID
  • Channel
